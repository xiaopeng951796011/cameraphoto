# EXIF Photo Frame

A clean Next.js app that reads photo EXIF metadata and renders an Apple-inspired white photography card with Canvas.

## Features

- Upload a photo and read EXIF metadata in the browser
- Extract camera maker, model, lens, focal length, aperture, shutter speed, and ISO
- Edit all caption text manually
- Render a white photo frame with rounded image corners and soft shadow
- Download the generated frame as a PNG
- Responsive layout for desktop and mobile

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
```
